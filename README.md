# LyonJS meetup social trailer videos generator

Welcome to your Remotion project!

## Commands

**Install Dependencies**

```console
pnpm i
```

**Start Preview**

```console
pnpm start
```

**Upgrade Remotion**

```console
pnpm run upgrade
```

**Format code with prettier**

```console
pnpm format
```
