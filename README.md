# LyonJS meetup social trailer videos generator

Welcome to your Remotion project!

## Commands

**Install Dependencies**

```console
pnpm i
```

**Start Preview**

```console
pnpm start
```

**Upgrade Remotion**

```console
pnpm run upgrade
```

## Available Trailer to render

- Sponsor

```console
yarn sponsor
```

- Meetup / Location + Date

```console
yarn meetup
```

- Talk & Speakers

```console
yarn talk
```
