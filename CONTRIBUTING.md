# Contribution documentation

## Commands

**Install Dependencies**

```console
pnpm i
```

**Start Preview**

```console
cd packages/remotion-components
pnpm start
```

**Upgrade Remotion**

```console
cd packages/remotion-components
pnpm run upgrade
```

**Format code with prettier**

```console
pnpm format
```
